# Quick Visual Guide - Where to Find Recycle Bin 🗑️

## Navigation Flow

```
App → Bottom Navigation → Profile Tab (Last icon) → Recycle Bin
```

## Step-by-Step Access

### 1. Open the App
You'll see the bottom navigation with tabs:
- 🏠 Home/Dashboard
- 📊 Statistics  
- 💰 Accounts (or Map for Owners)
- 👤 **Profile** ← Go here!

### 2. Tap Profile Tab
The Profile tab is the last icon in the bottom navigation

### 3. Scroll to "Preferences" Section
Look for the section titled **"PREFERENCES"**

### 4. Find "Recycle Bin"
You'll see these options:
- ⚙️ Settings (Theme, Notifications, Account)
- 🗑️ **Recycle Bin** (Deleted items (90 days)) ← This one!
- ℹ️ About (Privacy, Terms & App Version)

### 5. Tap "Recycle Bin"
The Recycle Bin screen will open showing:
- Info banner about 90-day retention
- Filter chips (All, Students, License, etc.)
- List of deleted items with:
  - Name and image
  - Deletion date
  - Days remaining before auto-deletion
  - Restore and Delete buttons

## In Recycle Bin

Once inside, you can:
- ✅ View all deleted items
- 🔍 Filter by type (Students, License, etc.)
- ↩️ Restore items (within 90 days)
- ❌ Permanently delete items
- 🧹 Clean up expired items

## Quick Access Tips

**Frequently Used**: Keep the Profile tab easily accessible in your bottom navigation

**Icon Recognition**: Look for the red trash icon (🗑️) in the Preferences section

**Subtitle Hint**: Remember "Deleted items (90 days)" subtitle helps identify it
